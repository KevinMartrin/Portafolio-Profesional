**[EN]**

I participated as a researcher in CIDESI, at PIIT Nuevo Leon, Mexico. I worked in the diagnosis of performance of CNC machinery for high-precision machined parts using IoT devices.

- Extraction of relevant variables from a non-relational MongoDB database
- Data analysis reports per machine using R.
- Creation of dashboards for each machine using real-time data.
