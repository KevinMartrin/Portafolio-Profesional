# [EN] Topological Data Analysis for Fuel Consumption and Carbon Footprint Reducing in Large Vehicle Fleets

Academic project in collaboration with a fuel transport company, utilizing Topological Data Analysis (TDA) techniques, such as the Mapper algorithm and Takens’ embedding, to uncover patterns and trends undetectable through traditional methods.

## 🎯 Objective
Identify trends and subtle characteristics to support the development of strategies for reducing a vehicle fleet's carbon footprint, without compromising process efficiency or productivity.

## 📦 Project Deliverable
- 📖 [Findings and Results Report:](https://github.com/KevinMartrin/Portfolio/blob/main/projects/TDA_for_fuel_consumption/TDA%20for%20fuel%20consumption%20optimization.pdf) A comprehensive report detailing the techniques used, their theoretical foundations, and the results derived from the partner company's dataset.

## 🛠️ Tools and Technologies
- Mapper Algorithm: Used to identify relationships and clusters within high-dimensional data.
- Takens' Embedding: Used to identify periodicity and underlying dynamics in variables.
- Python (Kmapper): For implementing the Mapper algorithm and Takens’ embedding, and visualizing them as simplicial complexes/graphs.
- Git and  GitHub: For version control and project hosting.

_Completed in collaboration with university colleagues._


# [ES] Análisis Topológico de Datos para la Reducción de Consumo de Combustile y Huella de Carboto en Flotas de Vehículos

Proyecto académico en colaboración con una empresa de transporte de combusstible, utilizando técnicas de análisis topológico de datos (TDA) como el algoritmo de Mapper y el encaje de Takens para hallar patrones y tendencias no apreciables con métodos tradicionales.

## 🎯 Objetivo
Encontrar tendencias y características difíciles de notar con técnicas usuales, que ayuden a desarrollar estrategias para la reducción de emisiones de carbono de una flota de vehículos, sin afectar la eficiencia y productividad de los procesos.

## 📦 Entregable del Proyecto
- 📖 **[Reporte de hallazgos y resultados](https://github.com/KevinMartrin/Portfolio/blob/main/projects/TDA_for_fuel_consumption/TDA%20for%20fuel%20consumption%20optimization.pdf):** Reporte que reúne las técnicas utilizadas, sus fundamentos teóricos y sus resultados, a partir de una base de datos suministrada por la empresa asociada.

## 🛠️ Herramientas y Tecnologías
- Algoritmo de Mapper: Para encontrar relaciones y patrones en datos de alta dimensión.
- Encaje de Takens: Para identificar periodicidad y dinámicas subyacentes en variables.
- [Python (Kmapper)](https://github.com/KevinMartrin/Portfolio/blob/main/projects/TDA_for_fuel_consumption/TDA_Mapper_Implementation.ipynb): Para ejectuar el algoritmo de Mapper y encaje de Takens y representarlos en forma de grafos.
- Git y GitHub: Como sistema de control de versiones.

_Hecho en colaboración con compañeros universitarios._
