# Green Logistics: Transport Routing Optimization for Reforestation 
## [EN]

Published academic article on [Applied Mathematics in Industry](https://www.springerprofessional.de/en/applied-mathematics-in-industry/51900042) (Springer) in 2026.

### 🎯 Objective
Minimize the distance and time associated with reforestation plan for a Mexican reforestal organization.

### ↗️ Results
- **Mathematical Model:** [Mixed-integer linear programming](Code/OrderingRoutes.gms) applied to the optimization of the Vehicle Routing Problem (VRP).
- **Heuristic Model:** A [heuristic](Code/HeuristicSolution.ipynb) to solve the problem almost optimally, with a fraction of the total time.

### 🛠️ Tools and Technologies
- GAMS (Solver)
- Python (Pandas, Numpy)
- Jupyter Notebooks
- Git y GitHub (version control)

*Made in collaboration with university partnerts*


## [ES]

Artículo académico publicado en [Applied Mathematics in Industry](https://www.springerprofessional.de/en/applied-mathematics-in-industry/51900042) (Springer) en 2026.

### 🎯 Objetivo
Minimizar la distancia y tiempo asociado de un plan de reforestación de una organización reforestal mexicana.

### ↗️ Resultados
- **Modelo Matemático:** [Programación lineal entera mixta](Code/OrderingRoutes.gms) aplicada en la optimización de un Problema de Enrutamiento de Vehículos (VRP).
- **Modelo Heurístico:** Una [heurística](Code/HeuristicSolution.ipynb) para resolver el problema casi óptimamente, con una fracción del tiempo total.

### 🛠️ Herramientas
- GAMS (Solver)
- Python (Pandas, Numpy)
- Jupyter Notebooks
- Git y GitHub (control de versiones)

*Hecho en colaboración con compañeros universitarios.*
